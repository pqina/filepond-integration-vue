# FilePond Integration with Vue

This template should help get you started developing with [FilePond](https://filepond.com) on [Vue](https://vuejs.org/) in [Vite](https://vite.dev).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
